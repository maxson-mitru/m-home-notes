# m-home-notes

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容均以静态 HTML 形式存放，不依赖特定后端服务，可直接通过浏览器打开或部署到任何静态托管平台。

## 项目简介

这是一个以 HTML 页面为主要内容的存档仓库。每个页面均独立存在，彼此之间没有强耦合关系，便于单独浏览、引用或分享。仓库本身不针对任何特定域名或网站，仅作为统一存放与版本管理的容器。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面文件，每个文件代表一个单独的页面内容。
- `assets/` — 存放页面所引用的公共资源，如样式表、脚本、图片等（如有）。
- `index.html` — 仓库根目录下的简单导航页，用于快速浏览所有已归档页面（可选）。

页面命名建议使用简洁的英文或拼音，避免特殊字符，确保在大多数操作系统和托管环境中正常工作。

## 页面归档说明

所有归档页面均遵循以下基本规则：

- 每个页面为一个完整的 HTML 文件，包含 `<!DOCTYPE html>` 声明及必要的元数据。
- 页面内引用的外部资源（如 CSS、JS、图片）优先使用相对路径或存放在 `assets/` 目录下。
- 页面内容不包含任何具体域名、网址、营销承诺或诱导点击元素。
- 页面标题与描述应准确反映该页面的实际用途。

如需新增页面，请直接在 `pages/` 目录下创建新 HTML 文件，并更新根目录下的导航页（如有）。

## 维护说明

本仓库由维护者根据实际需要不定期更新。欢迎通过 Issue 或 Pull Request 提交页面内容修正、样式优化或新增页面建议。

提交时请注意：

- 保持页面独立性与完整性，不依赖仓库外资源。
- 避免引入跟踪脚本、广告或外部第三方服务。
- 新页面内容需与仓库主题一致，不包含违规或敏感信息。
